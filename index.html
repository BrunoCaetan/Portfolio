TECNOLOGIAS / FERRAMENTAS

GITHUB

O GitHub é uma plataforma de hospedagem de código fonte que permite que desenvolvedores compartilhem, colaborem e controlem as versões de seu código. É amplamente utilizado para gerenciar projetos de software.

MYSQL

MySQL é um sistema de gerenciamento de banco de dados relacional amplamente usado. Ele fornece uma maneira de armazenar, organizar e recuperar dados, comumente utilizado em aplicativos da web e de negócios.

UNITY

Unity é um ambiente de desenvolvimento de software usado para criar jogos e aplicações interativas em 2D e 3D. É popular na indústria de jogos e realidade virtual/aumentada.

HTML

HTML é uma linguagem de marcação usada para criar páginas da web. Ela define a estrutura e o conteúdo de uma página, usando elementos e tags.

CSS

CSS é uma linguagem usada para estilizar páginas da web. Ele controla a aparência, o layout e o design das páginas, tornando-as visualmente atraentes.

C#

C# é uma linguagem de programação desenvolvida pela Microsoft. É usado para desenvolver uma ampla variedade de aplicativos, incluindo aplicativos Windows, jogos com a Unity e muito mais. É uma linguagem orientada a objetos e de propósito geral.
